Emoji Vault

Server emoji backup · GUI

Browse a server's custom emojis and download a full .zip backup — no Discord needed. Mirrors the bot's @SeekDeep emoji backup / list / count admin commands. Read-only: nothing here changes your server.

The Emoji Vault feature is off. Set SEEKDEEP_FEATURE_EMOJI_VAULT=on in your .env and restart the server to use it.
