How it works
  1. Make sure the ML libraries are installed — the banner turns green when Real-ESRGAN is ready.
  2. Pick a weight and hit Download (~64 MB). x4 is the general-purpose pick.
  3. It activates immediately — the next @SeekDeep upscale uses Real-ESRGAN instead of Lanczos. Restart the bot once for Discord to pick up the flag.
  4. No model or deps? Upscale silently falls back to the Lanczos resize, so it always works.
