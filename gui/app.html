    Add a model

    STEP 1 OF 4 · BACKEND
    Pick where this model runs. The first two stay on your machine; the last three reach out to a remote API and carry a privacy disclosure.
    HF · LOCAL DOWNLOAD Hugging Face snapshot_download into the cache. No daemon required, no key required.
    OLLAMA · LOCAL DAEMON Pulls a tag via POST /api/pull. Needs the Ollama daemon listening on :11434.
    OPENAI-COMPAT · BYK DeepSeek, OpenAI, Groq, OpenRouter, xAI Grok — anything that speaks the OpenAI Chat Completions shape. ⚠ prompts leave the local machine
    ANTHROPIC · BYK Native /v1/messages with x-api-key + anthropic-version headers. ⚠ prompts leave the local machine
    GEMINI · BYK Native :generateContent with x-goog-api-key. ⚠ prompts leave the local machine

    Pick a backend first.

    llama3.1:8b-instruct-q4_K_M qwen2.5:7b-instruct-q4_K_M mistral-nemo:12b-instruct-2407-q4_K_M phi-4:14b-q4_K_M (none installed · ‘pull new’ above will fetch)
    claude-opus-4-7 claude-sonnet-4-6 claude-haiku-4-5-20251001
    gemini-2.5-pro gemini-2.5-flash gemini-2.0-flash
    ⚠ Privacy disclosure. Prompts sent to this backend will leave your local machine. The remote provider receives the full prompt body in cleartext over TLS. Keep the key in .env only; never paste it into chat or commit it.

    Defaults vary by provider. DeepSeek = https://api.deepseek.com/v1 · Groq = https://api.groq.com/openai/v1 · Anthropic = https://api.anthropic.com/v1/messages · Gemini = https://generativelanguage.googleapis.com/v1beta.
