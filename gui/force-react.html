Force React

Reaction config · GUI

Configure the right-click Force React picker: how many reactions a user may add per message (the cap), an optional cooldown, and which server emojis the picker offers — so people can react with emoji they can't use in Discord directly.

Force React is off. Set SEEKDEEP_FEATURE_FORCE_REACT=on in your .env and restart the server to use it.
Cap is a hard, cumulative limit per user per message (1–20). Cooldown throttles repeat uses.
